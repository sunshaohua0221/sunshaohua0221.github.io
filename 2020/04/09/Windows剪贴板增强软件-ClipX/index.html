Windows剪贴板增强软件-ClipX

效率神器-只限windows

官网下载：http://bluemars.org/clipx/

能干啥

ClipX可以保存最近多次复制的内容，重复的内容不用来回复制了，支持文本、图片，而且复制的内容可以保存再本地，电脑重启后还能用

使用，图标右键

扩充剪贴板数量

最多支持保存1024条记录，正常人够用了

随时预览剪贴板内容

可以编辑剪贴板内容

数据本地存储

还得手动，自动存储就好了

快捷键设置

优点

  1. 免费
  2. 内存占用低
  3. 简单易用
  4. 侵入性小，不影响系统自带剪贴板
  5. 支持插件扩展
